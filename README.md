# dev-profile-card
A clean, responsive user profile card built with HTML and CSS. Designed to display a developer's contact information and social media links in a visually appealing layout.


## Live Demo
View Live Analog Clock🌐
https://ankamahjohnson.github.io/dev-profile-card/


## Features
- Responsive card layout centered on the page
- Profile photo with circular crop styling
- Contact details section with icons for email, phone, website, and location
- Social media links with branded icon buttons (Facebook, Twitter, Instagram, YouTube)
- Teal/dark header banner for visual contrast
- Clean typography and minimal design
- Fully responsive across all devices
